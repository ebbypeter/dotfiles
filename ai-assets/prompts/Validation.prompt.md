# Validation Prompt

This prompt is designed to encourage a thorough review and improvement of the initial response by generating and answering verification questions. It helps identify and address any potential errors, gaps, or issues in the original answer, leading to a more accurate and comprehensive final response.

```prompt
<Normal Prompt using any framework or structure>

Now follow these steps:
1. Provide your initial response
2. Generate 3-5 verification questions that would expose errors, gaps, issues in your answer
3. Answer each verification question independently
4. Provide your final revised answer based on the verification.
```
